# axis-bcd-filter

Фильтр входного потока данных с интерфейсом AXI4-Stream. Модуль пропускает только те входные слова, которые являются корректными в BCD представлении, для которых десятичное значение делится на 4 без остатка. Остальные слова отбрасываются

## Интерфейс

| Сигнал | Направление | Ширина | Описание |
|---|---|---|---|
| `clk` | in | 1 | Clock |
| `rst` | in | 1 | Сихронный ресет |
| `s_axis_tvalid` | in | 1 | Валидность входных данных |
| `s_axis_tready` | out | 1 | Модуль готов принимать  |
| `s_axis_tdata` | in | 8 | Входные данные |
| `m_axis_tvalid` | out | 1 | Валидность выходных данных |
| `m_axis_tready` | in | 1 | Готовность принимать выходные данные |
| `m_axis_tdata` | out | 8 | Выходные данные |

## Parameters

| Параметр | Знач. по умолчанию | Описание |
|---|---|---|
| `CHECK_BCD` | `1` | `1` = сбрасывать некорректные BCD (любой разряд > 9); `0` = считать все корректным BCD |
